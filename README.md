# 🏎 Turbo Race Extreme Bot

Премиальный гоночный бот для Telegram с использованием Mini App.

## Технологии
- **Backend**: Node.js, Telegraf 4.x, Express.
- **Frontend**: Canvas API, Tailwind CSS, Lucide Icons.
- **Telegram API**: WebApp SDK, Haptic Feedback.

## Запуск
1. `npm install`
2. Установите `BOT_TOKEN` и `WEBAPP_URL` в переменные окружения.
3. `npm start`
